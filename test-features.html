多维表格编辑器 - 功能测试指南

1. 单元格编辑功能 已修复
测试表格单元格的编辑功能，支持不同数据类型的编辑。
测试步骤：
  1. 双击任意可编辑单元格（如姓名、年龄、邮箱等）
  2. 观察是否出现相应的编辑器（文本框、数字框、下拉选择等）
  3. 修改内容后按 Enter 键或点击其他地方
  4. 检查数据是否正确保存和显示
2. 图片上传功能 已修复
测试图片上传和显示功能。
测试步骤：
  1. 右键点击任意单元格
  2. 选择"上传图片"选项
  3. 在弹出的模态框中选择或拖拽图片文件
  4. 点击"确认上传"按钮
  5. 检查图片是否正确显示在单元格中
3. 筛选功能 已修复
测试多条件筛选功能。
测试步骤：
  1. 点击工具栏中的"筛选"按钮
  2. 在右侧面板中点击"添加条件"
  3. 选择字段、操作符和筛选值
  4. 点击"应用筛选"按钮
  5. 检查表格是否按条件正确筛选
  6. 测试"清除筛选"功能
4. 分组功能 已修复
测试表格分组排序功能。
测试步骤：
  1. 点击工具栏中的"分组"按钮
  2. 在右侧面板中选择分组字段（如部门）
  3. 选择排序方式（升序/降序）
  4. 点击"应用分组"按钮
  5. 检查表格是否按选定字段正确排序
  6. 测试"清除分组"功能
5. 其他功能测试 正常工作
测试其他基础功能。
测试项目：
  1. 添加新行功能
  2. 删除行功能
  3. 数据保存和加载
  4. 数据导出（JSON/CSV）
  5. 键盘快捷键（Ctrl+S 保存、Ctrl+N 添加行等）
